## Cara Install Di Termux
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/alonecans/SelfCa
> cd SelfCa
> bash install.sh
> npm start/node main.js
```
## Cara Install Di Laptop
```bash
> git clone https://github.com/alonecans/SelfCa
> cd SelfCa
> npm i
> npm start/node main.js
```
